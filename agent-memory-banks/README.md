# Reserved for future agent roles

This directory contains memory banks for additional organizational agent roles as they are defined.

## Adding New Agents

When creating a new agent role:

1. Create `[role-name]-memory.md` in this directory
2. Update `AGENTS.md` with role definition
3. Add entry to `memory-bank/activeContext.md` agent tracking table
4. Document role in `roles/[role-name]/` directory
